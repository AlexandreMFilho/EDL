# Introduction to arquivos

  Programa que cria arquivos, faz a leitura e adiciona texto a um arquivo existente.

  Para executar basta digitar no terminal linux:

  >lein run  

  ou  
  >lein repl (use 'arquivos.core :reload)  
  >lein repl (-main)
